# Web Workspace

This folder is reserved for web application scope and documentation.

## Current State

The Next.js runtime currently lives at repository root:

- `app/`
- `components/`
- `lib/`
- `public/`

## Target State

Move all web runtime files into this folder (`web/`) in a dedicated migration.

## Web Stack

- Next.js (App Router)
- Tailwind CSS
- shadcn/ui
- Supabase client/SSR helpers

## Run (current root location)

```bash
npm run dev
```
